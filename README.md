# Web Browser

Web Browser using windows form application in c#. 

## Getting Started

Clone and open with Visual Studio 2019.

### Prerequisites

Use Visual Studio to open the project.

### Installing

1) Clone project
2) Open "Launcher" folder 
3) Open WebBrowser.exec
